# FTO Scheme Trainer

Simple practice app to get used to the FTO color scheme.

## Cool Features

- practice on rotated center cases, not just one fixed orientation
- color filters so you can train only the centers you want
- fast desktop input with left/right arrow answering
- mobile-friendly tap-to-start runs
- configurable training batches
- local stats for accuracy and average time per case
- quick case review in the stats panel, including reveal of the correct answer

Everything is stored locally in your browser, so the app keeps your settings and training stats between sessions.
